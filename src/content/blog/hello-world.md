欢迎来到我的个人网站！🎉

这个网站是我探索互联网世界的一个小角落。在这里我会分享：

- **技术文章**：编程、前端、工具链等
- **项目展示**：我做过的一些有趣的项目
- **随想随记**：关于技术和生活的思考

## 技术栈

本站使用 **Astro** 构建，托管在 **GitHub Pages**，主打一个轻量快速。

- 🎨 粒子背景特效 — tsParticles
- ⌨️ 打字机动画 — React 组件
- 🌙 暗色主题 — Tailwind CSS
- ⚡ 静态生成 — Astro
